Flagship Case Study

Market Narrative Storyboard Engine

A production-grade monorepo that aggregates market data, clusters financial narratives, validates technical setups, and generates deterministic media scripts for daily market storytelling.

Java 17 Spring Boot Next.js React SPA PostgreSQL Redis Docker

Built as a system, not a demo.

The backend is a modular Spring Boot monolith with clear internal boundaries for ingestion, clustering, technical validation, script generation, persistence, and administrative control.

The frontend deliberately separates public SEO delivery from private operational work: a Next.js portal publishes market digests, while a React admin studio supports authenticated editorial workflows. Simple where it can be, opinionated where it should be.
